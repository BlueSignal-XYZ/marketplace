# Feedback Log

Running log of corrections, preferences, and behavioral feedback. The assistant checks this file to avoid repeating mistakes.

**Format:** Log each entry under a dated heading with context about what happened and what should change.

---

## How to Use

When the assistant does something incorrectly or not to your preference:
1. Add a new entry below with the date and topic
2. Describe what happened and what the correct behavior should be
3. The assistant will check this file before acting on similar topics in the future

During weekly maintenance, recurring themes from this log get distilled into [[Knowledge/Training/Learned-Patterns]].
