# Product Images

This directory contains product mockup images for the BlueSignal sales portal.

## File Naming Convention

Product images should follow this naming pattern: `product-{product-id}.png`

## Required Images

| File Name | Product | Description |
|-----------|---------|-------------|
| `product-s-ac.png` | Shore Monitor AC | AC-powered algae control system |
| `product-s-sol.png` | Shore Monitor Solar | Solar-powered algae control system |
| `product-s-mon.png` | Shore Monitor Lite | Monitoring-only solar system |
| `product-smart-buoy.png` | Smart Buoy | Floating algae control platform |
| `product-smart-buoy-xl.png` | Smart Buoy XL | Research-grade floating platform |

## Image Specifications

- **Format**: PNG with transparent background (preferred) or JPEG
- **Dimensions**: 400x300px (recommended)
- **Aspect Ratio**: 4:3
- **Style**: Clean product mockup on transparent/white background
- **Quality**: High resolution, optimized for web

## Fallback Behavior

If an image is not found, the product card will display a placeholder icon.
Once you add the image file, it will automatically be displayed on the site.
